- **Bring to Front**: Move selected image on top of others

## ğŸ”§ Troubleshooting

### Images not appearing?
- **Check file format**: Ensure it's a supported image format
- **Check file size**: Very large files (>20MB) may load slowly
- **Refresh page**: Clear cache and reload

### Can't resize properly?
- **Use corner handles**: For proportional resize
- **Edge handles**: Resize in one direction
- **Check selection**: Ensure image is selected (blue border)

### Print doesn't match screen?
- **Check printer settings**: Ensure A4 paper size selected
- **No scaling**: Set print scaling to 100% or "Actual Size"
- **Margins**: Set to minimum or 0 for full A4 use

### Images overlap?
- **Use layer controls**: Bring desired image to front
- **Reposition**: Move images to avoid overlap
- **Resize**: Make images smaller if needed

## ğŸ¯ Best Practices

### For Best Print Quality

1. **Use high-resolution source images**
   - Minimum 300 DPI for print quality
   - Avoid web-sized low-res images

2. **Don't make images too large**
   - Pixelation occurs when scaling up too much
   - Keep within reasonable sizes

3. **Leave cutting margins**
   - Don't place images at exact edges
   - Leave 5-10mm margin for cutting

4. **Test print first**
   - Print on regular paper first
   - Check alignment and sizes
   - Adjust if needed, then print on good paper

5. **Use quality paper**
   - Photo paper for photos
   - Card stock for ID cards
   - Good quality A4 for documents
